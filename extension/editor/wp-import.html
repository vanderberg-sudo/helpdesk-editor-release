Before you start
  1. In your WordPress admin, go to Tools → Export → All content and download the XML file.
  2. Upload the XML file below.
  3. Choose Test (4 articles) first to verify the conversion looks right.
  4. Download the ZIP, extract it, and copy the folders into your Astro repo under src/content/docs/.
  5. When happy, run Full conversion for all 132 articles.
